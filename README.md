3. 목표로 하는 카지노봇 제작 (진행중)
3-1 칩 저장, 수정 및 출력기능 (완료)

3-3 승패 기록기능 추가 (완료)
3-4 승패 출력기능 (진행중)
3-5 추가 카지노 게임 (진행중)
